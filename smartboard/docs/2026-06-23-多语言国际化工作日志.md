## 📊 8 个图表组件国际化

### 基本图表
- **BarChart**：排序（自然/升序/降序）、堆叠/分组切换、标签按钮、指标选择提示
- **HorizontalBarChart**：同上
- **LineChart**：面积/纯线切换、平滑/折线切换、无数据提示
- **DoughnutChart**：标签按钮、无数据提示
- **HistogramChart**：自动选项、无数据提示

### 高级分析图表
- **TimeseriesChart**：周期切换（月/季/年）、图表标题、系列名（实际值/趋势/预测）、信息面板（最新/值/环比/同比/下期预测）、展开/收起明细、CSV 下载
- **DecileChart**：图表标题、分析指标标签、系列名（合计/数量）、展开/收起明细、CSV 下载
- **ClusterChart**：X 轴/Y 轴标签、图表标题、系列名（聚类 N/聚类中心）、汇总行（pts/Center）、展开/收起明细、CSV 下载

## 📋 三组明细表表头国际化

### 时序分析
周期 / 实际值 / MA3（含 title）/ 环比% / 同比% / 趋势 / 预测 → Period / Actual / MA3 / MoM% / YoY% / Trend / Forecast

### 十分位分析
分组 / 数量 / 合计 / 平均 / 范围 → Group / Count / Sum / Avg / Range

### 聚类分析
标签 / 聚类列 → Label / Cluster
汇总行：`607 个 · 中心` → `607 pts · Center`
明细行：`聚类 1` → `Cluster 1`

## 🧰 Toolbox 双语化

`chart-options.ts` — `buildToolbox()` 全部提示双语：

| 原文 | English |
|------|---------|
| 下载 | Save Image |
| 数据 | Data |
| 数据表 / 关闭 / 刷新 | Data Table / Close / Refresh |
| 还原 | Restore |
| 📋 复制表格 / ✅ 已复制 | Copy Table / Copied |
| 类别 / 数值 / 占比 | Category / Value / Ratio |
| 维度 | Dimension |
| 聚类 | Cluster |
| 序号 | # |

同步机制：`setToolboxLocale()` — Vue 应用由 `main.ts` watch 触发，独立 HTML 由 `initDashboard()` 触发。

## 📦 独立渲染器（entry.ts）
