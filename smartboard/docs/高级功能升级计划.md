## 二、现有图表优化清单

### 2.1 柱状图（BarChart）— P1

| 问题 | 现状 | 优化 |
|------|------|------|
| 标签显示 | 无数据标签 | 柱顶显示数值 |
| 排序 | 自然顺序 | 支持升/降序切换 |
| 参考线 | 无 | 支持平均值参考线 |
| 颜色 | 固定12色 | 支持按维度着色 |

### 2.2 环形图（DoughnutChart）— P1

| 问题 | 现状 | 优化 |
|------|------|------|
| 类别过多 | 全显示，标签重叠 | 超过8类自动合并为"其他" |
| 中心空洞 | 空白 | 显示总计数值 |
| 交互 | 无点击 | 点击扇区下钻/高亮联动 |
| 标签 | 仅图例 | 扇区上显示百分比 |

### 2.3 直方图（HistogramChart）— P2

| 问题 | 现状 | 优化 |
|------|------|------|
| 分箱数 | 自动 √N | 支持手动调节分箱数（滑块） |
| 叠加曲线 | 无 | 叠加正态分布/KDE 曲线 |
| 统计信息 | 无 | 显示均值、中位数、标准差 |

### 2.4 折线图（LineChart）— P1

| 问题 | 现状 | 优化 |
|------|------|------|
| 面积填充 | 无 | 支持切换面积图模式 |
| 标记点 | 无 | 自动标注最大值/最小值 |
| 指标数 | 单指标 | 支持多指标叠加对比 |
| 数据点 | 无圆点 | 悬停时显示数据点 |

### 2.5 通用增强 — P1

| 问题 | 方案 |
|------|------|
| 无导出按钮 | 每个图表右上角添加「下载PNG」按钮 |
| 无全屏 | 双击图表卡片全屏查看 |
| 无排序控制 | 柱状图/水平柱图增加排序方向按钮 |
| 暗色模式 | 部分图表 tooltip 在暗色下对比度不足 |

#### 1.1 相关性热力图（Correlation Heatmap）

**分析价值**：一键看清所有数值指标的相互关系。

```
            revenue  cost  orders  ad_spend  returns
revenue      1.00   0.95   0.98    0.72     -0.15
cost         0.95   1.00   0.93    0.68     -0.10
orders       0.98   0.93   1.00    0.70     -0.12
ad_spend     0.72   0.68   0.70    1.00      0.05
returns     -0.15  -0.10  -0.12    0.05      1.00
```

**技术方案**：
- 算法：皮尔逊相关系数矩阵
- 图表：ECharts Heatmap + visualMap
- 色阶：红（正相关）→ 白（无关）→ 蓝（负相关）

#### 1.2 帕累托图（Pareto Chart）

**分析价值**：比十分位更直观地展示"二八效应"。

```
柱状（降序） + 累计折线（第二Y轴）
→ 一眼看出前几项贡献了 80%
```

**技术方案**：
- 双 Y 轴：左轴柱状值，右轴累计百分比 0~100%
- 80% 参考线
- 复用现有 aggregate 逻辑

#### 1.3 异常值检测（Outlier Detection）

**分析价值**：自动发现数据中的离群点。

```
IQR 方法：
  Q1 = 25百分位, Q3 = 75百分位
  IQR = Q3 - Q1
  异常下界 = Q1 - 1.5×IQR
  异常上界 = Q3 + 1.5×IQR

输出：在现有图表上用散点/颜色高亮异常数据点
```

**技术方案**：
- 新增 `computeOutliers()` 函数到 `analysis.ts`
- 在各图表 option 中增加 markPoint 标注
- 数据表异常行高亮

- **趋势线**（长期方向）
- **季节分量**（周期性重复）
- **残差**（随机噪声）

#### 2.2 贡献度瀑布图

```

  - 华北下滑 5万
  + 华中增长 15万
  ─────────────
今年营收 200万
```
