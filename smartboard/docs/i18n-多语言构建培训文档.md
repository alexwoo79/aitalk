# SmartBoard 多语言（i18n）构建文档

## 1. 架构概览

```
src/i18n/
  index.ts          ← vue-i18n 插件配置，导出 LOCALES 列表
  zh-CN.ts          ← 中文语言包（~200 键）
  en-US.ts          ← 英文语言包（~200 键）

src/main.ts          ← 注册 i18n 插件 + 监听语言切换同步 toolbox
src/App.vue          ← 右上角语言切换下拉框 + localStorage 持久化

src/core/chart-options.ts  ← setToolboxLocale() + buildToolbox() 双语
src/save/entry.ts           ← 独立渲染器，内置 t() 函数
templates/result_template.html ← 注入 __I18N__ JSON + __DATA__.locale
```

**两套渲染路径：**

| 路径 | i18n 方案 |
|------|----------|
| Vue 应用（开发/生产） | `vue-i18n` 插件，`useI18n()` → `t()` |
| 保存的 HTML（独立渲染） | 模板注入 `__I18N__` JSON，entry.ts 内置 `t()` 查找 |

## 2. 语言文件结构

`src/i18n/zh-CN.ts` 和 `en-US.ts` 使用相同的嵌套键结构：

```ts
export default {
  common: { close, reset, search, save, cancel, label, ... },
  steps: { upload, config, result },
  upload: { title, subtitle, dropHint, ... },
  config: { title, saveAll, resetAll, chartType, ... },
  dashboard: { timeSlice, resetFilter, dataTable, ... },
  chart: { timeseries, decile, cluster, series: {...}, detailTable: {...} },
  chartType: { bar, line, doughnut, ... },
  kpiFormat: { global, number, currency, ... },
  datePreset: { last3Months, last6Months, ... },
  theme: { switchDark, switchLight },
}
```

### 添加新键的规则：
1. **两个文件必须同步添加** — zh-CN.ts 和 en-US.ts 的键结构完全一致
2. **键名使用 camelCase**，按功能模块分组
3. **支持参数化**：`'{n} records'` → `t('key', { n: 5 })` → `"5 records"`

## 3. Vue 组件中使用

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n'
const { t, locale } = useI18n()
</script>

<template>
  <!-- 简单文本 -->
  <span>{{ t('common.save') }}</span>

  <!-- 带参数 -->
  <span>{{ t('dashboard.recordsCount', { filtered: 100, total: 200 }) }}</span>

  <!-- 属性绑定 -->
  <input :placeholder="t('config.titlePlaceholder')" />
</template>
```

## 4. 独立渲染器（entry.ts）的 i18n

entry.ts 无法使用 vue-i18n，采用轻量方案：

### 4.1 模板注入
`saveDashboard()` 在保存时将当前语言包序列化为 JSON 注入 HTML：

```ts
// DashboardView.vue - saveDashboard()
.replace(/\{\{LOCALE\}\}/g, locale.value)
.replace(/\{\{I18N_JSON\}\}/g, JSON.stringify(locale.value === 'zh-CN' ? zhCN : enUS))
```

### 4.2 模板中的变量
```html
<!-- result_template.html -->
var __DATA__ = {
  ...
  locale: '{{LOCALE}}'        // 'zh-CN' | 'en-US'
};
var __I18N__ = {{I18N_JSON}}; // 完整语言包 JSON
```

### 4.3 entry.ts 中的 t() 函数
```ts
// 全局消息对象（从模板注入）
declare const __I18N__: Record<string, any>
let MSG: Record<string, any> = {}

// 点路径查找：t('chart.series.actual') → MSG.chart.series.actual
function t(path: string, params?: Record<string, string | number>): string {
  const keys = path.split('.')
  let val: any = MSG
  for (const k of keys) { if (val == null) break; val = val[k] }
  let result = (typeof val === 'string' ? val : path) as string
  if (params) {
    for (const [k, v] of Object.entries(params)) {
      result = result.replace(`{${k}}`, String(v))
    }
  }
  return result
}

// 初始化时设置
export function initDashboard(data: DashboardData) {
  MSG = typeof __I18N__ !== 'undefined' ? __I18N__ : {}
  setToolboxLocale(data.locale || 'zh-CN')
  ...
}
```

## 5. ECharts Toolbox 双语

`chart-options.ts` 中的 `buildToolbox()` 生成 ECharts 工具栏（下载/数据视图/还原），需要双语：

```ts
// 模块级 locale 变量
let _toolboxLocale = 'zh-CN'
export function setToolboxLocale(locale: string) { _toolboxLocale = locale }

export function buildToolbox(): Record<string, any> {
  const zh = _toolboxLocale === 'zh-CN'
  const L = {
    saveImage: zh ? '下载' : 'Save Image',
    dataView: zh ? '数据' : 'Data',
    restore: zh ? '还原' : 'Restore',
    // ...
  }
  // 使用 L.* 构建 toolbox feature
}
```

**同步机制：**
- Vue 应用：`main.ts` 中 `watch` i18n locale 变化 → 调用 `setToolboxLocale()`
- 独立 HTML：`initDashboard()` 中根据 `data.locale` 调用 `setToolboxLocale()`

## 6. 语言切换流程

```
用户点击右上角下拉框
  → App.vue: locale.value = 'en-US'
  → localStorage.setItem('locale', 'en-US')
  → vue-i18n 自动更新所有 t() 返回值
  → watch 触发 setToolboxLocale('en-US')
  → 所有图表 toolbox 提示切换为英文
```

刷新页面时：
```
main.ts onMounted
  → localStorage.getItem('locale')
  → 恢复上次选择的语言
```

## 8. 常见问题

### Q: 添加新页面/组件如何国际化？
1. 在 `zh-CN.ts` 和 `en-US.ts` 中添加新键
2. 组件中 `import { useI18n } from 'vue-i18n'` + `const { t } = useI18n()`
3. 模板中用 `{{ t('模块.键名') }}` 替换硬编码中文

### Q: 保存的 HTML 语言不对？
检查 `saveDashboard()` 中的 `locale.value` 是否正确传递。保存的 HTML 语言 = 保存时的当前语言。

### Q: entry.ts 中的字符串如何翻译？
使用 `t('key')` 函数（与 Vue 组件中相同的键路径）。entry.ts 在 `initDashboard()` 时从 `__I18N__` 全局变量加载消息。

### Q: 图表 toolbox 按钮提示没切换？
确认 `setToolboxLocale()` 已被调用。Vue 应用中由 `main.ts` 的 `watch` 自动处理；独立 HTML 中由 `initDashboard()` 处理。
