# 筛选条件 — 一页速查

## 运算符

| 运算符 | 含义 | 适用 |
|--------|------|------|
| `=` | 等于 | 全部列 |
| `!=` | 不等于 | 全部列 |
| `>` `<` `>=` `<=` | 比较 | 数值列 |
| `in` | 在列表中 | 文本列 |
| `~` `～` | 包含子串 | 文本列 |

## 连接符

| 符号 | 含义 |
|------|------|
| `&` | AND（且）|
| `|` | OR（或）|

## 兼容

- 全角 `～` = 半角 `~`
- `REGION` = `region`（大小写不敏感）
- `in` 分隔符不限：`,` `，` `、` `;` 空格

## 典型表达式

```
region = 华东
channel != 代理商
revenue > 100000
region in 华东,华南,华北
product_line ~ A
product_line ～ B          ← 全角～
PRODUCT_LINE = A产品线      ← 大小写

region = 华东 & channel = 线上
region = 华东 | region = 华南
region in 华东,华南 & revenue > 50000 & returns < 10
```
