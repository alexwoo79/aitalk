**目标**：

```
  无统计                    均值: 58,200  中位数: 52,100
```

- `src/core/chart-options.ts` — `buildHistogramOption` 增加 `binCount` 参数

**统计信息计算**（新增到 `chart-options.ts`）：
```typescript
function computeDistributionStats(values: number[]) {
  const n = values.length
  const mean = values.reduce((a,b) => a+b, 0) / n
  const sorted = [...values].sort((a,b) => a-b)
  const median = n % 2 ? sorted[Math.floor(n/2)] : (sorted[n/2-1] + sorted[n/2]) / 2
  const variance = values.reduce((s,v) => s + (v-mean)**2, 0) / n
  const std = Math.sqrt(variance)
  const skewness = values.reduce((s,v) => s + ((v-mean)/std)**3, 0) / n
  return { mean, median, std, skewness }
}
```

**工作量**：50 行，50 分钟。

```typescript
function downloadChart(chartRef: VChart) {
  const url = chartRef.chart.getDataURL({ type: 'png', pixelRatio: 2 })

  link.href = url

}
```

```
│ ├────────────────────────────────┤  │
```

**功能要点**：

| 功能 | 说明 |
|------|------|
| 展开/收起 | 点击 📋 按钮切换表格显示 |
| 自动聚合 | 表格数据 = 图表使用的聚合结果（不是原始行） |
| 排序 | 表头点击排序列 |
| 复制表格 | 一键复制为 TSV 格式，粘贴到 Excel/Google Sheets |
| 导出CSV | 下载为 .csv 文件 |

**各图表对应的表格数据**：

| 图表 | 表格列 | 表格行 |
|------|--------|--------|

| 折线图 | 日期 + 各指标 | 每个时间点一行 |

| 时序分析 | 周期 + 值 + MA3 + 环比 + 同比 | 每个周期一行 |
| 十分位 | 分位 + 条数 + 总和 + 均值 + 范围 | D1~D10 |
| 聚类 | 点标签 + X + Y + 所属簇 | 每个点一行 |

**ChartDataTable 组件设计**：

```typescript
// Props
interface Props {
  columns: string[]           // 表头列名
  rows: (string | number)[][] // 数据行
  visible: boolean            // 展开/收起
}

// Emits
emit('close')                 // 收起表格

// 功能
- 表格渲染（原生 <table>）
- 表头点击排序
- 复制按钮：navigator.clipboard.writeText(tsvString)
- 导出CSV：Blob + download link
```

**clipboard 复制实现**：

```typescript
function copyAsTSV(columns: string[], rows: (string|number)[][]) {
  const header = columns.join('\t')
  const body = rows.map(row => row.join('\t')).join('\n')
  const tsv = header + '\n' + body
  navigator.clipboard.writeText(tsv).then(() => {
    toast.show('已复制 ' + rows.length + ' 行数据')
  })
}
```

**数据获取方式**（以柱状图为例）：

